# College Bus Tracker - Frontend Prototype

This is a minimal static frontend prototype for a college bus tracking homepage built using only HTML and CSS.

Files:
- [index.html](index.html) — homepage with header, CSS-only menu, cards, and footer
- [styles.css](styles.css) — styles and responsive layout using CSS-only interactions

How to run:
1. Open `index.html` in your browser (double-click or right-click → Open with...).
2. For local development with auto-reload, serve the folder with a static server (e.g. `npx serve` or the VS Code Live Server extension).

Notes:
- The navigation toggle is implemented without JavaScript (checkbox + CSS selectors).
- Card actions link to static sections. To add dynamic maps or booking flows you'll need JS and backend APIs.

Next steps you might want:
